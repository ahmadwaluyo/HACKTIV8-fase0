TASK 2 :
Kamu diminta untuk membuat sebuah algoritma dan pseudocode untuk menghitung keliling sebuah lingkaran dimana hasil perhitungan akan ditampilkan.
Secara matematis rumus keliling lingkaran adalah 2 x pi x jari-jari
1. ALGORITMA
- DEKLARASI VARIABLE YANG DIGUNAKAN : KELILING LINGKARAN, PI , JARI-JARI;
- TETAPKAN VARIABLE PI BERNILAI KONSTANTA  = 22/7
- TETAPKAN VARIABLE JARI-JARI LINGKARAN DENGAN NILAI KONSTANTA = 14
- TETAPKAN NILAI KELILING LINGKARAN DENGAN KALKULASI 2 x PI x JARI-JARI LINGKARAN
- PANGGIL NILAI VARIABLE PI DENGAN NILAI JARI-JARI LINGKARAN
- HITUNG KELILING LINGKARAN 2 x 22/7 x 14 = 88 
- TAMPILKAN NILAI VARIABLE KELILING LINGKARAN



2.PSEUDOCODE
- DECLARE VARIABLE CIRCLE : CIRCUMFERANCE , PI, RADIUS 
- ‘SET’ VAR pi AS 22/7
- ‘SET’ RADIUS AS 14
- DEFINE CIRCUMFERANCE with ‘CALCULATATION’ =  2 x pi x RADIUS
- 'CALCULATE' 2 x 22/7 x 14 = 88
- DISPLAY CIRCUMFERANCE
